# Torment the Obelisk

Browser-based simulator and calculator project for **Idle Obelisk Miner**.

The project starts with a new/updated **Archaeology Sim** and is intended to grow into a multi-page web app with several simulators and calculators.

## Tech baseline

- Vite
- React
- TypeScript
- React Router
- StackBlitz / browser-first development

## Development

```bash
npm install
npm run dev
```

## Initial scope

- Home page
- Empty Archaeology page
- Clean folder structure for later separation of data, simulation logic, UI components, and routes
